About:
What is this?

DBcloud is a community that extracts structured knowledge from federated infrastructures, such as testbeds, and makes it freely available using Semantic Web and Linked Data standards. Inspired by DBpedia, the database currently contains over 3.3 million statements about the Fed4FIRE and GENI federations and their resources, with the potential to grow by many times this amount.

While the main focus in its initial developments have been set on Future Internet experimentation within federated testbeds, further fields of application include the management of Interclouds, the data collection and actuation in the Internet of Things or the orchestration of Virtualized Network Functions within Edge Computing environments for 5th generation mobile networks.

Open-Multinet:

At the heart of our knowledge base is the Open-Multinet semantic information model. Built upon a solid base of earlier research in semantic descriptions, it is used to describe all resource information.
